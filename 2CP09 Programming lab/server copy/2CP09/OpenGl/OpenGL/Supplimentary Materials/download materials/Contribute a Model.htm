Contribute a Model


     We are always on the lookout for more models! If you have a large geometric model and can obtain the appropriate permission to allow its public distributed, please send e-mail to Greg Turk. Noticeable gaps in the archive include terrain data, buildings, and models with color and/or texture information.

     What is a "large" model? For the purposes of this site, we have set a lower limit on model size based on the Stanford Bunny. The Bunny contains roughly 70,000 triangles, and no polygonal model at this site will contain fewer polygons. All of the models currently in the archive are polygonal, but we may consider including other kinds of models if they are large and are of high quality.
